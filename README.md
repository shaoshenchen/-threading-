# 使用说明

***

1. 客户端分别输入**用户名**，点击连接，并接受来自服务端确认消息；
2. 客户端选择**好友**，编辑消息内容，点击发送（支持微软输入法表情🙂）；
3. 点击清空清除屏幕内容。
